# Sobre a EAP
<p align="justify">Segundo o PMBOK 5ª edição, a Estrutura Analítica do Projeto (EAP) é “O processo de subdivisão das entregas e do trabalho do projeto em componentes menores e mais facilmente gerenciáveis.” .</p>
